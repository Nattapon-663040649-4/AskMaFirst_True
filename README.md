Project Overview
AskMaFirst is a mobile app feature designed to enhance financial safety for young users. Built on a TrueMoney Wallet–style interface, it allows parents to monitor and control their children’s spending while still enabling independent usage. The project was developed as part of a hackathon and focuses on secure, real-time transactions and parental approval workflows.

Key Features
Family Linking: Invite family members by phone number and assign Parent/Child roles
Spending Limits: Parents can set maximum transfer amounts per transaction for children
Transaction Monitoring: Parents can view transaction history in real time
Parental Approval System: Child transactions above the limit trigger a notification to the parent Parent must approve within 5 minutes; otherwise, the transaction is automatically rejected
Secure Backend: Firebase messaging used for real-time notifications and data synchronization

Tech Stack
Frontend: Flutter
Backend: Node.js, Express
Database & Messaging: Firebase
Version Control: GitHub
